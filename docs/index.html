Architecture

Four small services, one job each

Run them all with one docker compose up. Redis holds live goal state so the gateway can restart without losing a running goal.

gateway-rs
RUST · :9001 · the brain
Where the agents live and the goal pipeline runs. Parses your goal with an LLM, walks it through the state machine, and runs two background loops — a sentinel that checks positions every 5 min and a yield scheduler that hunts better APY every 30 min.
keymaster
RUST · :9100 · the vault
Owns your private keys and is the only thing that ever signs. The gateway never sees a key — it asks keymaster to swap, and keymaster checks the wallet's spend cap, enforces a slippage floor, signs, and broadcasts. Keys are encrypted at rest (AES-256-GCM + Argon2id).
nullboiler
ZIG · :9090 · the workflows
A DAG workflow engine with its own SQLite store. The gateway registers its agents as workers and can dispatch multi-step analysis pipelines through it, each step feeding the next.
dashboard
PREACT · :3000 · the cockpit
Create goals in plain English, watch every running goal live, browse wallet balances and the tx log, and hit the big red kill switch if everything's on fire.
The swarm

Nine agents, each a focused prompt

They don't run continuously — each is a small, single-purpose LLM wrapper invoked as the goal moves through its steps. Tap any agent to see its role and a snippet of its actual prompt.

Guardrails

Layers between an LLM and your money

The design goal: nothing can lose you more than you explicitly authorized. In order of defense —

① Wallet caps

Every wallet has a lifetime spend ceiling. Keymaster refuses to sign anything that would push it over the cap.

② Capital clamp

Even if the LLM hallucinates a giant number, the gateway clamps capital to 90% of the wallet cap before the goal starts.

③ Risk gate

Every candidate passes the risk agent first. HIGH risk — low TVL, unverified token, weird pair — means no trade. Try the "rug check" task above.

④ Slippage floor

Every swap carries an on-chain minimum-out from a live quote, so a trade can't be sandwiched to zero.

⑤ Kill switch

One flag halts the whole swarm — orchestrators and the goal pipeline both. Flip the toggle in the demo to feel it.

⑥ Dry-run mode

Go through every step including building the calldata, but never broadcast. Test your prompts without spending gas.

It's still your money, and this is early software. Vespra points an LLM at a live wallet — start on a testnet, run in dry-run first, watch the logs, and never trust it with more than you'd be okay losing.
